# Expense Tracker

A simple browser-based expense tracker with Indian rupee formatting.

## Features

- Add, edit, delete, search, and filter expenses
- View total spent, this month's spend, budget left, and largest category
- See category spending in a chart
- Save data in the browser using localStorage
- Export expenses as CSV

## Run Locally

Open `index.html` in a browser, or serve the folder with:

```bash
python -m http.server 5173
```

Then visit:

```text
http://127.0.0.1:5173/
